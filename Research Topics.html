Meta tags

Meta tags are used to provide information about the document to search engines and other web crawlers. They are not visible to the user but are used by search engines to index the document and display relevant information in search results.

URL

The url input type in HTML is specifically designed for capturing web addresses (URLs) from users. It offers a few advantages over a basic text input, such as built-in validation for URL format. Here's how you can utilize the url input type in a form: To use the url input type, you simply set the type attribute of an input tag to url.

What is the use of meta tag with attribute name and value viewport name="viewport"?

The meta tag with attribute name and value viewport is used to control the zooming and scaling of a web page. It is used to set the initial zoom level when a web page is first loaded.
